# CSharp (C#)

## Create Solution

```bash
    # Create new Solution
    dotnet new sln --name MySolution
    
    # Create new Project
    dotnet new console --output Projects/VivConsole
    
    # Solution Add Priject
    dotnet sln add Projects/VivConsole # (Remove) $ dotnet sln remove Projects/VivConsole
    
    # Get Solution Project list
    dotnet sln list
```
